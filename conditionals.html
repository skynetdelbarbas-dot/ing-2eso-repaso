Conditionals

Zero, first, second, third y mixed conditionals.

📖 Teoría rápida

0: Zero — verdades generales
If + Present Simple, Present Simple
If you heat ice, it melts.

1st: Primera — situaciones específicas / futuro probable
If + Present Simple, will + inf
If you study, you will pass the exam.

2nd: Segunda — hipótesis improbables
If + Past Simple, would + inf
3rd: Tercera — hipótesis imposibles (pasado)
If + Past Perfect, would have + p.p.
Mixed: combinan tiempos
If you study, you pass. — Cero
If you study, you will pass. — Primera
If you studied, you would pass. — Segunda
If you had studied, you would have passed. — Tercera
