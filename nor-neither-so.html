Inicio / Nor / Neither / So

Nor / Neither / So

Expresa acuerdo con frases afirmativas y negativas.

📖 Teoría rápida

SO + auxiliar + sujeto → acuerdo positivo.
"She likes pizza." → "So do I."

NEITHER / NOR + auxiliar + sujeto → acuerdo negativo.
"She doesn't like pizza." → "Neither do I." / "Nor do I."

El auxiliar debe coincidir: is → so am I · can → so can she · has → so has he · does → so does he
I love chocolate. → So do I.
I don't like spiders. → Neither do I.
