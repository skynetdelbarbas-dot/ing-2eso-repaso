🏢 Places (lugares): classroom, library, canteen, gym, playground, laboratory, office, sports field

✈️ Travel — Viajes

🚗 Transport: car, bus, train, plane, bike, taxi, ferry, underground / metro, tram, coach, motorbike
🏨 Accommodation: hotel, hostel, campsite, B&B, apartment, guesthouse, resort
🔁 Travel verbs: book, reserve, pack, arrive, depart, catch (a train), miss (a flight), check in, explore, visit, travel, go on holiday
🧳 Travel vocab: ticket, passport, luggage, suitcase, map, guide, journey, trip, flight, boarding pass, destination, souvenir
📍 Directions: turn left/right, go straight on, cross the road, at the corner, opposite, next to, between

🍔 Food & Drink — Comida y bebida

🍽️ Meals (comidas): breakfast, lunch, dinner, snack, dessert, brunch
🥦 Food groups: fruit (apple, banana, orange), vegetables (carrot, tomato, onion), meat (chicken, beef, pork), fish, dairy (milk, cheese, yoghurt), grains (bread, rice, pasta)
👨‍🍳 Cooking verbs: cook, boil, fry, bake, roast, grill, chop, mix, peel, pour, add, stir, cut

🔀 Phrasal Verbs

Verbos compuestos: verbo + partícula. El significado cambia completamente.

⬆ UP

get up (levantarse)· wake up (despertarse)· give up (rendirse / dejar)· pick up (recoger)· look up (buscar información)· turn up (subir el volumen / aparecer)· grow up (crecer)· make up (inventar / maquillar / reconciliarse)· clean up (limpiar, recoger)· end up (terminar haciendo algo)

⬇️ DOWN

turn down (bajar volumen / rechazar)· put down (dejar / anotar)· write down (apuntar)· calm down (calmarse)· break down (averiarse / derrumbarse)· slow down (reducir la velocidad)· settle down (establecerse / sentar cabeza)

↗️ OFF

turn off / switch off (apagar)· take off (quitarse ropa / despegar)· put off (posponer)· go off (dispararse / estropearse / sonar alarma)· set off (ponerse en camino)· show off (presumir)· pay off (pagar deuda / dar resultado)

↩️ OUT

find out (descubrir / averiguar)· run out of (quedarse sin)· go out (salir / apagarse)· come out (salir / publicarse)· sort out (solucionar / ordenar)· work out (resolver / hacer ejercicio)· figure out (averiguar, entender)· check out (echar un vistazo / pagar y marcharse)· start out (empezar, comenzar)

🔁 IN

come in (entrar)· get in (entrar al coche / llegar)· give in (rendirse / ceder)· fill in (rellenar un formulario)· check in (registrarse en hotel / facturar)· believe in (creer en)· join in (unirse a / participar)

🎯 Word Formation

Cómo cambiar una palabra añadiendo prefijos y sufijos para formar nuevas palabras.

📌 Sufijos — cambian el tipo de palabra

-er / -or persona que hace algo: teach → teacher, act → actor
-tion / -sion acción o proceso: inform → information, decide → decision
-ness cualidad (adj → noun): happy → happiness, kind → kindness
-ment resultado o estado: agree → agreement, enjoy → enjoyment
-ly forma adverbio (adj → adv): quick → quickly, careful → carefully
-ful lleno de: help → helpful, beauty → beautiful, care → careful
-less sin: home → homeless, care → careless, help → helpless
-able / -ible que se puede: comfort → comfortable, access → accessible
-ous lleno de / carácter: danger → dangerous, fame → famous
-ish aproximación o nacionalidad: child → childish, Spain → Spanish
-y que tiene cualidad: sun → sunny, rain → rainy, health → healthy
-al relativo a: music → musical, nature → natural, education → educational

📌 Prefijos — cambian el significado

un- no / inverso: happy → unhappy, friendly → unfriendly, pack → unpack
im- / in- / ir- / il- no (antes de p, b, r, l): possible → impossible, correct → incorrect, regular → irregular, legal → illegal
dis- no / inverso: agree → disagree, appear → disappear, honest → dishonest
re- de nuevo / otra vez: write → rewrite, do → redo, build → rebuild, cycle → recycle
mis- mal / incorrectamente: understand → misunderstand, spell → misspell, lead → mislead
over- demasiado / exceso: cook → overcook, sleep → oversleep, eat → overeat
pre- antes: view → preview, pay → prepay, heat → preheat
inter- entre: national → international, net → internet, act → interact
Ejemplos en contexto:
She's a very creative (create) person.
It was unfair (fair) to blame him.
The film was useless (use) — don't watch it.
I need to rewrite (write) my essay.
